## 4. 서비스 팝업 (핵심)

### 동작 방식
- 페이지 로드할 때마다 **항상** 팝업 표시
- ✕ 버튼 / 닫기 버튼 / 오버레이 클릭 → 팝업 닫힘
- 닫힌 후 → 우하단 FAB 버튼 표시 ("🔧 AI 개발 상담 문의")
- FAB 클릭 → 팝업 재표시

### 핵심 훅 카피

```
🍉 수박 먼저 잘라드립니다. 달면 사세요.

완성이 100이라면, 핵심 기능 10%를 Vercel에 배포해드립니다.
실제 URL로 접속해서 직접 써보세요.
쓸만하면 나머지 90% 진행 — 아니면 그냥 무료로 가져가세요.

저는 신규라 아직 리뷰가 없습니다. 그래서 먼저 드립니다.
```

### 전략적 배경
```
❌ 일반 서비스: "믿고 맡겨주세요" → 계약 → 결제 → 결과
✅ 이 서비스:  상담 → 무료 제작 → Vercel URL → 직접 써보기 → 그다음 결정
```

신규 프리랜서라 리뷰·후기가 없음 → 리스크를 내가 먼저 짊어지는 방식으로 신뢰 구축

### 4단계 진행 방식

| 단계 | 내용 |
|------|------|
| ① 무료 상담 | 원하는 서비스 말씀해주세요 |
| ② 우선순위 TOP 10 | 기능 목록 함께 정하기 (혼자 정하면 나중에 딴 소리 나옴) |
| ③ 10% Vercel 배포 | 핵심 기능 무료 제작 → Vercel 실제 배포 |
| ④ 직접 써보고 결정 | URL 접속 → 쓸만하면 계속 / 아니면 무료로 가져가기 |

### 서비스 가격표

| 패키지 | 가격 | 기간 | 내용 |
|--------|------|------|------|
| 단순 버그 | 7만원 | 당일 | 오류 수정 1건 + 원인 설명 |
| 버그 + 원인 문서 | 20만원 | 3일 | 수정 + 왜 고장났는지 문서 1장 |
| 10기능 설계 + 구현 | 50만원 | 5일 | 전체 기능 설계 + 핵심 기능 구현 |

### 결제 기준
- 20만원 이하: 카카오페이 선불 100%
- 20만원 이상: 착수금 50% 선불 → 완료 후 잔금 50%
